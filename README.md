## Installation
### cURL
```sh
sudo curl -sL "https://raw.githubusercontent.com/poodypai/nepse-price-checker/main/nepse" -o /usr/local/bin/nepse
sudo chmod +x /usr/local/bin/nepse
```
## Usage
```sh
nepse STOCK_SYMBOL
eg. nepse GBIME
